  ---
  三個公式的關係

  c = m^e mod n           ← 加密操作（怎麼用公鑰）
  m = c^d mod n           ← 解密操作（怎麼用私鑰）
  (m^e)^d mod n = m       ← 數學定理（為什麼上面兩個能配對）

  前兩個是怎麼用，第三個是為什麼能用。

  把第一個代入第二個就看得很清楚：

  c = m^e mod n
  m = c^d mod n = (m^e)^d mod n

  所以第三個公式就是前兩個能成立的數學保證。

  ---
  還缺一塊：怎麼造出 e 和 d

  公式裡的 e 和 d 不是隨便挑的，它們必須滿足一個關係：

  e × d ≡ 1  (mod φ(n))

  其中 φ(n) = (p-1)(q-1)        ← 歐拉函數，需要 p、q 才能算

  直觀理解：e 和 d 是一對「互為逆元」的數，乘起來在模 φ(n) 下等於 1。

  關鍵點：算 d 必須知道 p 和 q，這就是為什麼分解 n 等於拿到私鑰。

  ---
  完整的核心其實是四個

  1. n = p × q                    建鑰匙（造牆）
  2. e × d ≡ 1 (mod φ(n))         建鑰匙（造門和鑰匙）
  3. c = m^e mod n                加密
  4. m = c^d mod n                解密

  背後的數學定理:
     (m^e)^d mod n = m            歐拉定理保證上面成立

  ---
  一句話

  ▎ 三個公式是 RSA 的「使用說明」，背後還藏著一個造鑰匙的公式 e × d ≡ 1 (mod φ(n)) —— 整個 RSA 就是這幾條數學關係搭起來的。
